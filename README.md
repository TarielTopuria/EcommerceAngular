# EcommerceAngular

Angular ecommerce demo (FakeStore API) located in the `ecommerce-angular/` folder.

## Run locally

```bash
cd ecommerce-angular
npm install
npm start
```

Open `http://localhost:4200/`.

## Documentation

See `ecommerce-angular/README.md` for:

- business logic (auth, admin allowlist, cart persistence)
- demo credentials
- environment configuration
